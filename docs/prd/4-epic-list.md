# 4. Epic List

## Epic 1: Adaptive Core Engine & Project Foundation
Establish project infrastructure (CI/CD, documentation, dev tooling) and deliver the foundational workflow engine end-to-end — YAML parse to workflow result with branching, retry, and LLM invocation.

## Epic 2: Compositional Primitives
Expand the primitive library with the full set of compositional building blocks: chat, output-generator, call-agent, guardrail, tool, and structured output enforcement.

## Epic 3: Observability & Integration
Add production-grade observability (OpenTelemetry tracing, lifecycle hooks) and framework integration (FastAPI handler, SSE streaming) to make workflows deployable and monitorable.

## Epic 4: Adaptive Execution Patterns (Post-MVP)
Advanced control flow: reflection loops, goal-oriented execution, parallel fan-out, and circuit breaker for provider-level resilience.

## Epic 5: Agent Autonomy & Safety (Post-MVP)
Enterprise-grade features: human-in-the-loop approval, model tier selection, PII tokenization, and state persistence/checkpoints.
